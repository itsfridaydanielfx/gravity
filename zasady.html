Jak liczone są punkty

Scaled bodyweight = bodyweight ^ 0.67. Każdy raw score jest mnożony przez scale x10, a potem przez mnożnik ćwiczenia. Finalny wynik ćwiczenia jest zaokrąglany do 1 miejsca po przecinku.

Mnożniki ćwiczeń

Pull-up 1.6 • Dip 1.2 • Muscle-up 2.0 • Bench 1.7 • Squat 1.1 • Deadlift 0.9 • OHP 2.2 • Biceps 2.8

Jaki wynik wpisujemy

  • Bench, squat, deadlift, OHP, biceps: 1RM lub ustalony PR.
  • Pull-up, dip, muscle-up: dodatkowy ciężar na jedno strict powtórzenie.
  • Overall wymaga kompletu wszystkich 8 bojów.

Standard wykonania

  • Pull-up: pełen wyprost, broda nad drążkiem, bez kippingu.
  • Dip: pełen lockout, odpowiednia głębokość, bez odbijania.
  • Muscle-up: strict weighted standard.
  • Bench / squat / deadlift / OHP / biceps: jeden spójny standard dla wszystkich.

Remisy

Ten sam wynik = to samo miejsce. Kolejna pozycja przeskakuje naturalnie, np. 1, 2, 2, 4.

Ważne

Jeśli chcesz zmienić metodologię, edytuj config.js. Nie koduj mnożników na sztywno w HTML.
