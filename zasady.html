Methodology
Rules
⚙️

How points are calculated

Scaled bodyweight = bodyweight ^ 0.67. Every raw score is multiplied by readability scale x10 and then by exercise multiplier. Final exercise score is rounded to 1 decimal place.

Exercise multipliers

Pull-up 1.6 • Dip 1.2 • Muscle-up 2.0 • Bench 1.7 • Squat 1.1 • Deadlift 0.9 • OHP 2.2 • Biceps 2.8

What result should be entered

  • Bench, squat, deadlift, OHP, biceps: 1RM or fixed PR standard.
  • Pull-up, dip, muscle-up: added weight for one strict rep.
  • Overall ranking requires all 8 lifts completed.

Movement standards

  • Pull-up: full elbow extension, chin clearly over bar, no kipping.
  • Dip: full lockout, proper depth, no bounce.
  • Muscle-up: strict weighted standard, no kipping unless you decide otherwise.
  • Bench / squat / deadlift / OHP / biceps: use one clear internal standard and keep it consistent for everyone.

Ties

Same score = same rank. The next rank is skipped naturally. Example: 1, 2, 2, 4.

Important

If you want to change methodology later, edit config.js. Do not hardcode multipliers inside HTML.
